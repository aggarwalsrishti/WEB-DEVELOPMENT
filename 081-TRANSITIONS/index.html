BASIC TRANSITION

Hover the box to see it grow in size and change color smoothly over 2 seconds.

TRANSITION ON A SINGLE PROPERTY

Hover the box → only width expands gradually in 3s, color changes instantly.

TRANSITION TIMING FUNCTIONS

linear → same speed from start to end.
ease → default, starts slow → speeds up → slows down.
ease-in → slow at start.
ease-out → slow at end.
ease-in-out → slow at start and end.



TRANSITION WITH DELAY

Hover the box → transition waits 1s before starting, then expands smoothly.
