            position: value;
            top: 10px;
            left: 20px;

Possible Values:

static → Default positioning (normal document flow).
relative → Positioned relative to its normal position.
absolute → Positioned relative to the nearest positioned ancestor.
fixed → Positioned relative to the viewport (stays fixed while scrolling).
sticky → Acts like relative until a scroll threshold is reached, then “sticks” like fixed.
